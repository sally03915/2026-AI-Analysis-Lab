## 📅 2개월차 : LangChain & 웹 배포 (엔터프라이즈 아키텍트 트랙)

### 1주차. LangChain 핵심 구조 학습

* **Ch 1. LangChain 생태계와 LCEL의 이해**
* 왜 쌩(Raw) 코드가 아니라 프레임워크인가? LangChain 도입 이유
* LCEL(LangChain Expression Language)의 파이프 연산자(`|`) 메커니즘


* **Ch 2. 템플릿 바인딩과 가변 인자 제어**
* `PromptTemplate`과 `ChatPromptTemplate`을 활용한 동적 입력 제어


* **Ch 3. [실습] 파이썬 history 코드를 LangChain Memory로 전환**
* `ConversationBufferMemory`와 `RunnableWithMessageHistory` 활용
* 대화 이력 관리 코드의 리팩토링 및 간소화



### 2주차. RAG 기반 문서 검색 및 요약 자동화

* **Ch 1. RAG(Retrieval-Augmented Generation) 아키텍처 기본**
* Document Loaders(PDF, TXT, CSV)와 Text Splitters(청킹 전략)
* 임베딩(Embedding) 모델의 원리와 Vector 스토어(Chroma DB) 연동


* **Ch 2. 기업 비용 절감을 위한 Prompt Caching 전략**
* 중복 컨텍스트 비용을 줄이는 캐싱 매커니즘 이론


* **Ch 3. [실습] 대용량 보고서 요약 및 Advanced RAG 튜닝**
* 다량의 사내 문서를 청킹·임베딩하여 핵심 요약본 추출
* Re-ranking 알고리즘을 활용한 유사도 검색 쿼리 고도화



### 3주차. 사내 문서 검색 챗봇 제작 및 Streamlit Cloud 배포

* **Ch 1. Streamlit 고급 상태 관리 (`session_state`)**
* 화면이 리프레시되어도 끊기지 않는 웹 데이터 유지 기법


* **Ch 2. RAG 백엔드와 웹 UI 컴포넌트 결합**
* 2주차 RAG 파이프라인 엔진을 Streamlit 고급 레이아웃에 이식


* **Ch 3. [실습] Production 환경 배포 및 보안 셋업**
* GitHub 연동 및 Streamlit Cloud 무료 배포 프로세스
* API 키 유출 방지를 위한 `Secrets` 환경 변수 관리



### 4주차. 강의 교안 최적화 (개발자 vs 비개발자 시나리오)

* **Ch 1. 타겟 맞춤형 교육 난이도 밸런싱**
* 비개발자용 직관적 개념 시각화 vs 개발자용 아키텍처 분석


* **Ch 2. 실습 코드 리팩토링 및 주석 작업**
* 수강생들에게 배포할 에러 없고 직관적인 마스터 코드 패키징


* **Ch 3. [미션] 2트랙 분리형 강의 제안용 교안 초안 작성**
