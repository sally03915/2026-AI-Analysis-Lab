* HRD 담당자의 높은 오픈율과 회신율을 이끌어내는 B2B 콜드메일(Cold-Mail) 제목 작성법 및 제안서 첨부 템플릿을 익힌다.
* 최소 3곳 이상의 대형 교육 플랫폼에 공식 지원서를 제출하고, 실제 세일즈 파이프라인을 가동한다.

## 📖 3. 핵심 이론 집중 강의안 (50분 분량)

### 1) 메이저 교육 에이전시 공략의 핵심

* **에이전시의 니즈 파악:** 교육 플랫폼(패스트캠퍼스, 멀티캠퍼스 등)은 "강의를 잘하고 인기가 많으며, 기업 B2B 특판 영업에 즉시 투입할 수 있는 강사"를 갈구함.
* **제안서의 차별화:** 단순 이력서 제출이 아니라, 앞서 만든 **Streamlit 데모 웹 링크**와 **임원/실무진 투트랙 커리큘럼**을 함께 첨부하여 에이전시 담당자의 기획 고민을 한 번에 해결해 주어야 함.

---

### 2) 기업 HRD 담당자 대상 콜드메일 세일즈 공식

* **오픈율을 높이는 제목:** "사내 AI 도입" 같은 뻔한 제목 대신, [제안] 임직원 반복 업무 70% 단축 '생성형 AI 자동화 실무' 출강 제안 (데모 링크 포함)처럼 구체적인 임팩트를 제시.
* **콜드메일 구조:** 인사말 ➔ 기업이 겪는 문제 공감 ➔ 솔루션 요약 ➔ 포트폴리오 링크 ➔ 10분 미팅 제안(Call to Action).

```python
# ==============================================================================
# 🚀 4개월차 4주차: B2B 에이전시 지원 및 콜드메일 세일즈 자동화 템플릿
# ==============================================================================
# [수강생 안내] 본 코드는 기업 HRD 담당자 및 메이저 교육 에이전시(패스트캠퍼스, 멀티캠퍼스 등)에
# 즉시 발송하여 미팅 성사율을 극대화하는 고효율 콜드메일 템플릿과 지원서 데이터를 생성합니다.

# 1. 의존성 패키지 설치 (코랩 환경 실행 시 주석 해제)
# !pip install -q langchain-openai==0.1.20 langchain-core==0.2.33 python-dotenv==1.0.1

import os
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser

print("📦 B2B 세일즈 및 콜드메일 템플릿 실습 라이브러리 임포트 완료")

# API Key 설정 관리
if "OPENAI_API_KEY" not in os.environ:
    os.environ["OPENAI_API_KEY"] = "sk-proj-YOUR_MOCK_KEY_FOR_CLASS"

# ==============================================================================
# [실습 1 & 2] HRD 담당자 타겟 고성과 콜드메일 카피라이팅 생성 (강사 시간: 100분)
# ==============================================================================
def generate_cold_mail_template(target_company_type: str, portfolio_url: str) -> str:
    """
    기업 교육 담당자나 에이전시 매니저에게 발송하여 오픈율과 회신율을 높이는 
    B2B 콜드메일 본문 템플릿을 생성합니다.
    """
    print(f"📧 [콜드메일 세일즈] '{target_company_type}' 타겟 맞춤형 메일 초안 작성 중...")
    
    sales_engine = ChatOpenAI(model="gpt-4o-mini", temperature=0.2)
    
    mail_prompt = ChatPromptTemplate.from_messages([
        ("system", 
         "당신은 B2B 에듀테크 세일즈 전문가입니다. "
         "기업 HRD 담당자의 마음을 사로잡을 수 있는 직관적이고 설득력 있는 콜드메일 템플릿을 작성해주세요. "
         "지나친 광고성 어조를 배제하고, 실무 생산성 향상과 구체적인 데모 링크 확인을 유도해야 합니다."
        ),
        ("user", "타겟 대상: {target}\n포트폴리오 데모 링크: {link}")
    ])
    
    chain = mail_prompt | sales_engine | StrOutputParser()
    result = chain.invoke({
        "target": target_company_type,
        "link": portfolio_url
    })
    return result

# 콜드메일 생성 시뮬레이션
mock_target = "대기업 및 금융권 HRD 교육 담당 부서"
mock_link = "https://enterprise-rag.streamlit.app"

cold_mail_draft = generate_cold_mail_template(mock_target, mock_link)
print(f"\n📨 [생성된 B2B 콜드메일 템플릿 미리보기]:\n{cold_mail_draft}")


# ==============================================================================
# [실습 3] [미션] 메이저 교육 플랫폼 공식 지원 체크리스트 및 세일즈 가동 (강사 시간: 50분)
# ==============================================================================
def execute_b2b_launch_checklist() -> str:
    """
    4개월 대장정의 피날레로, 수강생이 즉시 실행해야 할 메이저 플랫폼 지원 체크리스트를 출력합니다.
    """
    print("\n🚀 [B2B 시장 런칭] 메이저 교육 플랫폼 공식 지원 체크리스트 로드 중...")
    
    checklist = """
    ==================================================
    [🎯 4개월차 최종 미션: B2B 세일즈 및 에이전시 런칭 체크리스트]
    ==================================================
    1. 메이저 에이전시 강사풀 등록 (최소 3곳 이상 지원 완료)
       - [ ] 패스트캠퍼스 (B2B 기업교육 파트너 강사 지원)
       - [ ] 멀티캠퍼스 / 휴넷 (테크 및 생성형 AI 전문 강사 풀 등록)
       - [ ] 이오플로우 / 크몽 엔터프라이즈 전문가 등록
    
    2. 디지털 포트폴리오 및 제안서 최종 검수
       - Streamlit 클라우드 데모 링크 3개 정상 작동 여부 확인
       - 4주차 패키징된 B2B 표준 견적서 및 커리큘럼 PDF 변환 완료
    
    3. 콜드메일 세일즈 가동
       - 타겟 기업 HRD 담당자 이메일 리스트 업 (최소 10개사)
       - 상기 생성된 맞춤형 콜드메일 및 제안서 일괄 발송 및 팔로업
    ==================================================
    """
    return checklist.strip()

launch_checklist = execute_b2b_launch_checklist()
print(f"\n{launch_checklist}")

print("\n🎉 [전체 커리큘럼 완성] 1~4개월차에 걸친 'AI 엔지니어 및 B2B 지식 창업가' 마스터 파이프라인 구축이 성공적으로 모두 완료되었습니다!")

```
